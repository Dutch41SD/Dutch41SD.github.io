IT Projects

TacoCat

 

The Goal: A palindrome checker built on strict data sanitization.

How it works: The JavaScript engine uses Regular Expressions (RegEx) to strip away all spaces and punctuation. It then converts the clean text to lowercase, splits it, reverses it, and compares it to the original input to verify a perfect match.

Loan Shark

 

The Goal: A financial calculator handling complex state management.

How it works: The script intercepts raw text inputs and converts them into strict numerical floats. It then applies standard amortization formulas to calculate monthly payments, while simultaneously running logic to guard against blank or invalid data.
